# NZP-Calc

Автономный браузерный расчёт физических объёмов и стоимости НЗП по данным SharePoint.

Основополагающий источник объёмов — одна или несколько книг «БД КФО» (`.xlsb`/`.xlsx`). Для каждого источника в `config.json` отдельно задаются ссылка, листы «КФО»/«ДК», строка поиска заголовков, ячейка даты обновления и резервные буквы колонок. При чтении приложение сначала ищет колонки по именам заголовков, поэтому их перемещение в книге не ломает настройку.

Расчёт поддерживает уровни KQ-1 и KQ-2 и два режима: без регламентов и с регламентами листа «ДК». В режиме регламентов применяются отдельные коэффициенты к «Факт СМР накопительно», «Принято СК» и «Принято ИД»; для последнего используется коэффициент этапа «Архив ИД». Лист «KQ» исходной книги считается контрольным и в расчёте не участвует.

Срез по дате использует «Дата внесения РД» и «Дата последнего факта СМР». В исходной книге нет отдельных дат для «Принято СК» и «Принято ИД», поэтому их точный откат на прошлую дату возможен только при наличии отдельной книги-среза за эту дату.
